CSS

HTML로 작성된 문서가 웹사이트에 표현되는 방법을 정해주는 스타일 시트 용어이다. Cascading Style Sheet을 줄여서 CSS라고 한다. 여기서 cascading은 상위요소의 스타일 속성을 자손요소에게 상속 시켜주는 모습이 DOM 트리구조에서 자치 폭포수 처럼 내려가는 모습을 닮았기 때문이다. CSS는 인라인 방식, style 태그 안에 작성하는 방법, 별도의 stylesheet를 이용하는 방법이 있다. 적용되는 우선 순위는 인라인 style > id > class> type 순으로 우선순위가 높다. 선택자를 통해 스타일을 입히며, .은 class, #는 id, 이 외에도 여러가지 선택자가 있다.
